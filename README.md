# Telegram Reel Downloader Bot 🎥

A Telegram bot that:
1. Accepts Instagram Reel links
2. Downloads the video
3. Adds a custom watermark
4. Sends back the processed video

## Setup

### 1. Clone repo

### 2. Install dependencies

### 3. Install FFmpeg
https://ffmpeg.org/download.html

### 4. Add your bot token

### 5. Run bot

## Notes
- Use a transparent PNG for watermark
- Works best with public reels
